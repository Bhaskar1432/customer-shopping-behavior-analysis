# customer-shopping-behavior-analysis
Data Analytics Project
Overview

This project demonstrates a complete end-to-end data analytics workflow. The goal is to analyze a dataset, extract insights, and present the findings through a dashboard and reports.

The project includes:

Loading and exploring the dataset using Python

Performing Exploratory Data Analysis (EDA)

Cleaning and preparing the data

Running analytical queries using PostgreSQL

Creating an interactive Power BI dashboard

Preparing a project report and presentation

This project highlights skills in data analysis, SQL, data visualization, and business insight generation.

Dataset

The dataset contains customer shopping behavior data used to analyze patterns in purchasing behavior.

Example features in the dataset include:

Customer ID

Gender

Age

Product Category

Purchase Amount

Payment Method

Purchase Date

The dataset is used to understand customer trends, spending behavior, and product preferences.

Tools and Technologies

The following tools were used in this project:

Python – Data loading, cleaning, and analysis

Pandas & NumPy – Data manipulation and preprocessing

Matplotlib / Seaborn – Data visualization for EDA

PostgreSQL – Running SQL queries and data analysis

Power BI – Creating interactive dashboards

Gamma – Creating presentation slides

Jupyter Notebook – Analysis and experimentation

Project Steps
1. Data Loading

The dataset was loaded into Python using Pandas for initial exploration.

2. Exploratory Data Analysis (EDA)

EDA was performed to understand the dataset and identify patterns such as:

Distribution of customer demographics

Spending behavior

Product category popularity

Purchase trends

3. Data Cleaning

Data preprocessing steps included:

Handling missing values

Removing duplicates

Fixing data types

Standardizing column names

4. SQL Analysis

The cleaned dataset was imported into PostgreSQL. SQL queries were written to analyze:

Top selling product categories

Customer purchase behavior

Sales distribution

Customer segmentation

5. Dashboard Development

An interactive dashboard was built in Power BI to visualize key insights such as:

Sales trends

Customer demographics

Category performance

Revenue analysis

6. Reporting

A detailed report was created summarizing the analysis and insights derived from the data.

7. Presentation

A professional project presentation was created using Gamma to communicate results and insights clearly.

Dashboard

The Power BI dashboard provides interactive visualizations that help explore the dataset easily.

Key dashboard features include:

Sales by product category

Customer demographic analysis

Purchase trends over time

Revenue distribution

The dashboard allows users to filter and interact with the data to gain deeper insights.

Results and Insights

Some key insights obtained from the analysis include:

Identification of high-performing product categories

Understanding of customer purchasing behavior

Key demographic segments contributing to sales

Trends in customer spending patterns

These insights can help businesses make data-driven decisions to improve marketing strategies and customer engagement
